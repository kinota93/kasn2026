### 年度初めにシステム側で準備すること

1. phpMyadmin上で新年度のデータベース「`kasn2026db`」に作成する

2. 設定ファイル`conf/config.inc.php`を編集する。
   - 対象データベースを切り替え
     - `'dbname' => 'kasn2026db' // 'kasn2025db'`
   - 日程なども併せて編集する
     - `$conf['app']['begin'] = '2026-4-08 10:00';`
     - `$conf['app']['end']   = '2026-4-15 18:30';`

3. 研究室テーブルと学生テーブルのデータを編集
   - `xt_student_utf8n.csv`
   - `xt_lab_utf8n.csv`

   注意：文字コードはBOMなしUTF8にする必要がある
　 TeraPadで「文字/改行コード指定保存(K) 
   - 文字コード：`UTF-8N`
   - 改行コード：`CR-LF`

4. 管理者でログインし、システム初期化メニュで上記のファイルをアップロード
